## 三、关键决策记录

| 决策 | 结论 |
|:----|:-----|
| **avg_salary_history 单位** | 统一为**元/月**（引擎 getBase() 直接取用，月缴=base×index×8%） |
| **base_rates 单位** | 统一为**元/月**（计发基数月均值） |
| **计发月数表** | 全国统一标准，各省无需独立配置，删除22省JSON覆盖 |
| **用户系统** | 不做（不需要保存计算记录） |
| **付费定价** | 详细报告 1元/次（企业资质已申请） |
