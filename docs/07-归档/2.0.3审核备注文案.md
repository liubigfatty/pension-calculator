# 小程序 2.0.3 提交审核备注文案

> 复制下面「版本更新说明」整段内容，粘贴到微信公众平台「提交审核 → 版本说明 / 审核备注」即可。

---

【版本更新说明】

一、小程序功能
本小程序为个人养老金测算工具。用户填写参保地、人员类型、参加工作时间与出生年月、平均缴费指数、个人账户余额等信息后，引擎按各省份政策测算每月可领取养老金，生成包含"基础养老金 / 过渡性养老金 / 个人账户养老金 / 增发"构成的明细报告，并支持"法定退休"与"弹性提前退休"两种方案的对比。

二、本次 2.0.3 更新内容
1. 修复报告分享长图（保存图片）时，底部说明与免责声明被截断、顶部标题文字重叠显示不完整的问题；现报告长图可完整、清晰展示全部测算结果与说明。
2. 同步校准 31 个省份的计发基数、社平工资、记账利率等数据，与各地人社 / 税务官方口径一致。
3. 优化付费报告解锁流程，改用微信虚拟支付"道具直购"能力。

三、关于付费功能（如实说明，便于审核）
报告中的"解锁完整报告"为一次性付费内容（1 元 / 次），采用微信虚拟支付"道具直购"模式，用于解锁详细测算明细与分享长图；属虚拟商品交付，不涉及任何实物发货、不涉及自动续费或订阅。
如审核需要体验付费流程，可在测试环境发起支付验证。
