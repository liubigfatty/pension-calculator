## 二、测试用例与结果

### 2.1 案例1：北京-女工人50岁（有过渡性养老金）✅

**测试目的**：验证过渡性养老金计算（G同+G实）

**输入参数**：
```
省份：北京
性别：女
退休类型：工人 50岁
出生年月：1973年10月
工作年月：1995年11月
退休年月：2023年10月
---
平均缴费指数：0.4584
个人账户余额：116362.74
实际缴费年限：21年
视同缴费年限：4.01年
1997年底前缴费：2.58年
```

**预期结果**：
```
基础养老金：    2394.17 元
个人账户养老金： 596.73 元
过渡性养老金：   139.27 元
总额：          3130.17 元
```

**实际结果**：✅ **与预期完全一致**

**结论**：通过 ✅
