| 参数构造 | ✅ | 字段名与云函数要求一致 |
| 结果处理 | ✅ | 正确解析 `res.result.data` |

测试页面调用代码示例（`miniprogram/pages/cloud-test/cloud-test.js`）：
```javascript
wx.cloud.callFunction({
  name: 'calculate',
  data: {
    province: 'beijing',
    cityType: 'prov',
    gender: 'male',
    identity: 'employee',
    genderType: 'male',
    birthDate: '1965-10',
    workStartDate: '1985-07',
    averageIndex: 1.2,
    personalAccount: 80000,
    extras: {}
  },
  success: (res) => {
    if (res.result && res.result.success) {
      this.setData({ result: JSON.stringify(res.result.data, null, 2) })
    }
  },
  fail: (err) => {
    this.setData({ error: err.message })
  }
})
```

### 3.2 参数格式对齐

| 参数名 | 小程序端格式 | 云函数端要求 | 对齐结果 |
|---------|----------------|----------------|----------|
| `province` | 小写字母，如 `beijing` | 小写字母，加载 `provinces/${province}.js` | ✅ 一致 |
| `birthDate` | `'1965-10'`（YYYY-MM） | 解析为 `birthYear`, `birthMonth` | ✅ 一致 |
| `workStartDate` | `'1985-07'`（YYYY-MM） | 解析为 `workYear`, `workMonth` | ✅ 一致 |
| `averageIndex` | 数字，如 `1.2` | `parseFloat()` | ✅ 一致 |
| `personalAccount` | 数字，如 `80000` | `parseFloat() \|\| 0` | ✅ 一致 |
