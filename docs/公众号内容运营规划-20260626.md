# 正元养老金测算 — 公众号内容运营规划

> 更新：2026-06-26

---

## 一、三条内容线

| 线 | 方向 | 节奏 |
|:--|:-----|:----:|
| 🟠 **31省养老金图鉴** | 每省一篇（数据+排名+案例+特殊政策） | 每周2-3篇 |
| 🔵 **退休决策指南** | 弹性退休/缴费年限/跨省对比等决策类 | 穿插 |
| 🟢 **政策解读热点** | 新基数公布、利率调整等 | 追热点 |

## 二、省份生产顺序

**S级**：吉林（已验证，优先）、辽宁、黑龙江（东北三省联动）  
**A级**：北京、上海、河南、河北、山东、广东、浙江（人口大省/一线）  
**B级**：四川、重庆、西藏、青海（特殊政策）  
**C级**：其他省份（按反馈热度）

## 三、首月排期

**第1周**：吉林篇 → 东北三省对比 → 缴费年限决策  
**第2周**：北京篇 → 延迟退休决策 → 河南篇  
**第3周**：50vs60退休对比 → 上海vs老家对比 → 灵活就业决策  
**第4周**：西藏高原补贴 → 跨省社保转移 → 15年vs30年对比
