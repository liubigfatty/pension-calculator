# 截图清单 - v2.0

> 提审时需上传功能截图，用于审核人员了解页面功能。

---

## 截图要求

| 要求 | 说明 |
|------|------|
| 图片格式 | PNG / JPG |
| 分辨率 | 750×1334 px（推荐） |
| 数量 | 3-5张 |
| 内容 | 展示核心功能页面 |

---

## 建议截图列表

| # | 页面 | 截图内容 | 优先级 |
|---|------|---------|--------|
| 1 | 首页 | 品牌展示+"开始测算"按钮 | 必选 |
| 2 | 缴费信息页 | 填写缴费指数和账户余额 | 必选 |
| 3 | 结果页 | 显示养老金计算结果（含金额卡片） | 必选 |
| 4 | 报告页 | 退休规划深度报告前2个模块 | 推荐（v2.0新功能） |
| 5 | 报告保存 | 保存到相册后的图片预览 | 推荐（v2.0新功能） |

---

## 操作提示

1. 使用微信开发者工具截图：在模拟器中打开对应页面 → 截图
2. 或使用真实设备截图：真机预览 → 安卓/iOS原生截图
3. 截图应包含完整的页面内容，不裁切
4. 截图应使用测试数据（如吉林省、月养老金 ¥3,882.11）
