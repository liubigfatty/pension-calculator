## 📋 截图要求

### 技术规范
- **格式**：PNG 或 JPG
- **比例**：手机竖屏（9:16 或 3:4）
- **分辨率**：建议 750×1334px 或更高
- **文件大小**：每张 < 2MB

### 内容要求
- 截图必须展示**真实功能界面**（不能用设计图代替）
- 界面内容要**完整**（不要只截一半）
- 数据要**真实**（用测试数据填充，不要用"测试"、"demo"等字样）
- 如果是真机截图，确保**没有微信通知栏遮挡**

---

## 🛠️ 如何获取截图

### 方法一：真机预览（推荐）
1. 打开微信开发者工具
2. 点击「预览」→ 生成二维码
3. 用手机微信扫码
4. 在真机上操作到对应页面
5. 截图（手机自带截图功能）
6. 发送到电脑，保存到本目录

### 方法二：开发者工具截图
1. 在开发者工具中操作到对应页面
2. 点击工具栏的「截图」按钮
3. 保存到本目录
