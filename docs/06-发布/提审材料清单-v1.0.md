## 📤 提审操作步骤

### 1. 上传代码
1. 打开微信开发者工具
2. 点击「上传」按钮
3. 填写版本号：`v1.0.0`
4. 填写项目备注：`首次提交，31省养老金计算，含弹性提前退休功能`

### 2. 提交审核
1. 登录微信公众平台（mp.weixin.qq.com）
2. 进入「管理」→「版本管理」
3. 点击「提交审核」
4. 填写审核信息：
   - **功能页面**：`pages/index/index`（审核人员从这个页面开始测试）
   - **测试账号**：无需提供（匿名使用）
   - **审核说明**：
     ```
     本小程序为养老金测算工具，覆盖全国31省。
     核心功能：选择参保地区 → 填写个人信息 → 设置缴费参数 → 查看养老金测算结果 → 分享给好友。
     支持法定年龄退休和弹性提前退休两种计算方式。
     详细测试步骤见附件《提审测试步骤-v1.0.md》。
     ```
5. 上传6张功能截图
6. 点击「提交审核」

### 3. 审核跟踪
- **审核周期**：1-3 个工作日（首次可能 3-7 天）
- **审核结果**：微信会发送通知（服务通知）
- **审核失败**：查看拒绝原因，修复后重新提交
