## 🔓 测试说明

**本项目无需登录，匿名使用，无测试账号。**

审核人员测试步骤：
1. 打开小程序 → 点击「开始测算」
2. 选择参保省份（如「吉林省」）
3. 填写个人信息（性别、出生年月、参加工作时间）
4. 选择退休方式（法定年龄退休 / 弹性提前退休）
5. 填写缴费信息（平均缴费指数、个人账户余额）
6. 点击「计算养老金」→ 查看结果
7. 测试分享功能（分享给好友 / 生成分享图）

## 🔒 隐私协议

**隐私协议页面**：`pages/privacy/privacy`

**隐私协议内容要点**：
- 不收集姓名、身份证号、手机号等个人敏感信息
- 所有输入数据仅存储在用户手机本地缓存
- 使用微信云函数进行测算，不持久化存储用户参数
- 用户权利：查阅权、更正权、删除权、撤回同意权

**首次进入弹窗**：已启用 `__usePrivacyCheck__: true`，首次打开小程序时会自动弹出隐私协议确认框。
