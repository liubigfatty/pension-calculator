## 📷 截图操作指南

### 方法1：微信开发者工具截图
1. 在微信开发者工具中运行小程序
2. 打开对应页面
3. 点击工具栏「截图」按钮（或 `Ctrl+Shift+P` → 输入「截图」）
4. 保存到 `docs/06-发布/审核材料-v1.0.5/截图/` 目录

### 方法2：真机调试截图
1. 点击「真机调试」
2. 用手机打开小程序，进入对应页面
3. 手机截图（电源键+音量键）
4. 发送到电脑，保存到上述目录

## ⚠️ 截图注意事项

1. **清晰度高**：文字可读，不模糊
2. **完整页面**：包含导航栏和底部（如有）
3. **真实数据**：用真实测算案例（如「吉林省，男，1970年6月生，1995年7月参加工作」）
4. **隐私脱敏**：如截图包含真实个人信息，需打码处理
5. **文件命名**：按序号命名，方便审核人员对照查看
