# 上海数据核对报告（2026-07-23）

> 方法：对照 `articles/上海市社平工资-现实调音师.md` 官方年表 + 官网核实（WebSearch：沪劳综发〔1996〕48号、上海本地宝历年社平、澎湃/中新网 2019 全口径公告）。
> 规则：上海"计发基数 = 上一年社平"。2019 年社保口径转向全口径，故 2020 年起计发基数取"上年全口径社平"，1997–2019 取"上年非私营全市职工平均工资"（即引擎 `AVG[year-1]` 序列）。

## 一、发现的问题

### 问题1：PROV_BASE[1995–2019] 整体错位（已订正）
原值 595–8181，严重违背"计发基数=上年社平"规则。例如 `PROV_BASE[2019]=8181`，但 2018 社平应为 9184（MD/引擎 AVG[2018]）。
**订正**：按规则重排 `PROV_BASE[year] = AVG[year-1]`（非私营序列），1997→3139 … 2019→9184，与引擎 AVG 序列完全一致。原 8201.1/7736.98/8181.09 等错位占位数据已删除。

### 问题2：AVG[1990–1995] 占位错误（已订正）
原值 195–249 元/月，量级明显偏低。官网核实：
- 沪劳综发〔1996〕48号：1995 年度全市职工月平均工资 = **773 元**；
- 国家统计局市区职工工资序列：1990–1994 ≈ 247 / 287 / 364 / 483 / 631 元/月。
**订正**：`AVG[1990–1995]` 改为 246.96/287.00/364.03/483.06/630.93/773，并标注为旧口径（与 1996+ 序列量级断层，仅作个人账户历史近似，不用于计发基数推算）。

### 问题3：PROV_BASE[2020]=9580 与 AVG[2019]=9643 的 63 差（非错误，已说明）
网核实证：2019 全口径社平 = 9580（澎湃新闻、中新网、上海本地宝一致引用上海统计局 114962元/年）。2020 为口径切换过渡年，**计发基数取全口径 9580**；引擎 `AVG[2019]=9643` 记录的是非私营旧序列，二者用途不同，属正常口径差，**非数据错误**，保留 9580。

## 二、未改动项（已核验正确）
- `PROV_BASE[2021–2025]` = 10338 / 11396 / 12183 / 12307 / 12434，与"上年全口径社平"完全一致（本地宝官方历年社平核验通过）。
- `AVG[1996–2024]` = 3139…12434，与 MD 上海人社局年表一致，2020–2024 与本地宝全口径逐值吻合。

## 三、遗留待补（推定）
- `PROV_BASE[1995]`、`PROV_BASE[1996]`：跨 1995/1996 口径断层（旧职工平均工资→新序列），无法从 `AVG[1994–1995]` 可靠推导，暂标"推定待补"。
- `AVG[1990–1994]`：旧口径国家统计局序列，标"推定"（MD 明确不列示该段）。
