# 公众号发文前自检清单（防"低创作度"限流）

> 适用场景：每篇发文前，照着勾一遍。4 大类全过再发；任一项"否"就先改再发。
> 背景：本号曾因"低创作度内容"被平台预警，被点名文章集中在四类问题——下面清单就是照这四类反推出来的。

---

## 一、同质化检查（别和自己的旧文撞车）

- [ ] 近 30 天发过同主题文章吗？__________（有/无）
- [ ] 本篇和近期文章**信息重复度是否 > 60%**？
  - 若 > 60% → **必须合并或差异化**，不要单独发。
  - ✅ 正确做法：把"领失业金养老怎么交"和"失业金怎么领"合成一篇《失业金全攻略》，删掉重复那篇。
  - ❌ 反面教材：本号《领失业金，养老保险还交吗？》与《吉林省失业保险金怎么领？领多少？领多久？》主题高度重叠，被判定同质化。

---

## 二、搬运 / 抄袭检查（政策条文要"翻译"，别照搬）

- [ ] 文章里**大段引用政策原文 / 别人内容**了吗？
  - 若引用 > 全文 40% → 必须加自己的解读、案例、本地化口径。
- [ ] 有没有**本地化加工**？（长春/吉林各地办理流程、本地电话、分档标准）
  - ❌ 反面教材：《4050补贴都谁能领》偏政策条文搬运，缺本地办理口径和自测，被判定搬运+信息量不足。
  - ✅ 改法：加"长春本地办理流程 + 你符不符合的自测表"。

---

## 三、信息量检查（裸数据 = 低价值，数据 + 解读 = 高价值）

- [ ] 是不是"一张表 / 一堆数字"就结束了？
  - 若是 → 必须补三层：**为什么涨、涨了多少、跟你有什么关系**。
- [ ] 有没有给读者**可操作的结论**？（算给你看 / 对比 / 建议）
  - ❌ 反面教材：《黑龙江省历年社平工资表（1992–2025）》只有裸表，无解读、无和养老金的关联，被判定信息量不足 / 低价值 AIGC。
  - ✅ 改法：加"近10年涨幅 + 社平涨对你退休金的影响测算 + 吉黑对比"，立刻变高价值原创。

---

## 四、低价值 AIGC 检查（AI 能辅助，但不能"代写到底"）

- [ ] 这篇是不是 AI 直接生成、几乎没人工深加工？
  - 若是 → 必须叠加：**原创观点 / 真实案例 / 互动提问 / 个人判断**。
- [ ] 读者读完能不能获得**信息增量或情绪价值**？
  - 信息增量：新数据、新角度、算给你看。
  - 情绪价值：替读者着急、帮读者避坑、说人话。
- [ ] 数据有没有**标注来源 + 说明口径**？（避免"看起来像编的"）

---

## 五、发前总评分（全"是"才发）

| 检查项 | 是否通过 |
|--------|---------|
| 1. 不与近期旧文同质化 | □ 是 □ 否 |
| 2. 非纯搬运，有本地化加工 | □ 是 □ 否 |
| 3. 不是裸数据，有解读+可操作结论 | □ 是 □ 否 |
| 4. AI 内容有人工深加工，有增量/情绪价值 | □ 是 □ 否 |

**4 项全"是" → 放心发。**
任意"否" → 先补那一项再发。

---

## 六、黄金公式（记住这一句就行）

> **数据 / 政策 + 你的解读 + 读者能得到的好处 = 平台爱推的内容**
>
> 缺任何一块，就容易被判"低创作度"。
